UI와 API, 문서

UI : 사용자의 의중을 시스템에 전달하는 것! ex. 버튼

Application Programming Interface API : 이미 만들어진 것을 웹 브라우저에 띄울 수 있도록 명령어를 사용한 것 ex. alert 명령문으로 경고창을 띄우기

레퍼런스 : 명렁어의 사전 튜토리얼 : 언어의 문법을 설명하는 것, 안내서

자바스크립트의 API
